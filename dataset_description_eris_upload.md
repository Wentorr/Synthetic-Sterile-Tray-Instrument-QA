# Dataset Description

## Overview

This synthetic dataset contains rendered sterile instrument tray inspection images. Each PNG shows a wrapped surgical tray with six numbered compartments, color-coded simplified instrument shapes, a sterility indicator strip, a two-digit tray tag, a checklist strip, an orientation arrow, and a visible seal-breach marker.

The dataset is a multimodal image question-answering benchmark. Each row pairs an image with a natural-language multiple-choice question and five answer options. Correct answers are generated from the hidden scene trace. Questions require visual inspection, spatial reasoning, counting, small-digit reading, color/shape matching, and robustness to glare, blue cast, dense clutter, and partial wrap occlusion.

The data is fully synthetic. It contains no real patient, hospital, instrument manufacturer, operating-room, or proprietary clinical data.

## File Structure

| File or Folder | Rows / Count | Description |
|---|---:|---|
| `questions.csv` | 448 | Raw QA table with labels and hidden evaluation groups. The prepare script splits this into public train/test and private answers. |
| `images/` | 56 PNG files | Rendered sterile-tray inspection images referenced by the `image` column. |

The prepare script copies PNGs into `public/images/`, creates a salted scene-level train/test split, remaps question IDs and image filenames to opaque values, places labeled rows into `public/train.csv`, places unlabeled rows into `public/test.csv`, and stores labels plus hidden evaluation groups in `private/answers.csv`.

## Public Columns After Preparation

| Column | Type | Description |
|---|---:|---|
| `id` | string | Opaque unique question id generated by the prepare script. |
| `image` | string | Relative path to the PNG under `public/`. |
| `query_type` | string | Question category. |
| `question` | string | Natural-language multiple-choice question. |
| `option_A` | string | Answer option A. |
| `option_B` | string | Answer option B. |
| `option_C` | string | Answer option C. |
| `option_D` | string | Answer option D. |
| `option_E` | string | Answer option E. |
| `answer` | string | Correct answer letter. Present only in `train.csv`. |

## Query Types

| Query Type | Description |
|---|---|
| `instrument_slot` | Identify which numbered compartment contains a named instrument. |
| `empty_slot` | Identify which compartment is empty. |
| `indicator_color` | Identify the sterility indicator strip color. |
| `tag_code` | Read the two-digit tray tag display. |
| `seal_breach` | Identify the zone containing the red wrap breach mark. |
| `orientation_arrow` | Determine which direction the orange orientation arrow points. |
| `instrument_count` | Count how many visible instances of a named instrument appear. |
| `checklist_order` | Determine which checklist step follows another in the strip. |

## Hidden Evaluation Groups

The private answers include hidden grouping columns used by the grader:

| Group | Values |
|---|---|
| `query_type` | The eight query types listed above. |
| `split_group` | `standard_holdout`, `ood_low_light`, `ood_blue_cast`, `ood_glare`, `ood_wrap_occlusion`, `ood_dense_clutter`. |
| `ood_axis` | `standard`, `low_light`, `blue_cast`, `glare`, `wrap_occlusion`, `dense_clutter`. |
| `layout_family` | `standard_grid`, `wide_grid`, `compact_grid`, `offset_grid`. |
| `lighting_condition` | `normal`, `low_light`, `glare`, `blue_cast`. |
| `clutter_level` | `low`, `medium`, `high`. |
| `task_family` | `inventory_check`, `sterility_check`, `text_reading`, `spatial_reasoning`, `sequence_memory`. |

The score rewards balanced performance across these hidden groups, so a solution that handles normal-light counting but fails on digit reading, occlusion, or seal-breach localization will not score well.

## Image Characteristics

- Resolution: 720x480 RGB PNG.
- Scene count: 56 tray images.
- QA rows: 8 questions per image, 448 total.
- Train/test split: salted scene-level split.
- Visual variation: layout family, low light, blue cast, glare, partial wrap occlusion, sterile wrapper speckles, and distractor instrument fragments.
- Important: the answer must be inferred from the image and question/options. Hidden scene traces are not present in the public data.
